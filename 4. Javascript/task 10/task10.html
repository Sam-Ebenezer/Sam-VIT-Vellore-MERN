Array methods

Check console for result!

Concat

Concatenate two strings

CopyWithin

used to copy the elements within the array to other elements in the array without changing array length

Map

Used to map each element in array to another element

Every

Used to check if every element in array passes a condition

Some

Used to check if atleast one element in array passes a condition

Filter

creates a new array from the original array where the elements in the new array all passes a condition

Find

find the first element that satisfies the condition in the array

FindIndex

find the index of the first element that satisfies the condition in the array

IndexOf

returns the first index at which a given element can be found in the array, or -1 if it is not present.

lastIndexOf

returns the last index at which a given element can be found in the array, or -1 if it is not present.

Push

adds an element to the end of the array.

Pop

removes the last element of the array.

Shift

removes the first element of the array.
